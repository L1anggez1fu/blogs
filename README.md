## Blogs

  基于Spring Boot框架开发

​	
## 开发环境

**Idea+JDK1.8+Tomcat**

​		
## 技术栈

### 前端
- Css框架： Semantic UI
- JS框架： JQuery
- MarkDown编辑器： Editor.md
- 目录生成： Tocbot
### 后端
- 框架： SpringBoot 2.2.4
- 模板引擎：Thymeleaf 3
- 持久层： Mybaits
- 构建： Maven
### 数据库
- MySQL 8.0.18
